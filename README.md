
# Building Height Estimation Tool

Urban Height AI is a simple web application that estimates **building heights** using **SAR (Synthetic Aperture Radar) and Optical satellite images**.
The project demonstrates the concept of **Multi-Sensor Fusion**, where two types of satellite data are combined to estimate urban building heights.
---
## 🚀 Features
- Upload **SAR Image**
- Upload **Optical Image**
- **Click on buildings** to select them
- Predict **building heights**
- Calculate **MAE and RMSE metrics**
- Show **Predicted vs Actual height chart**
---
## 🧠 How It Works
1. Upload SAR and Optical images  
2. Click on buildings in the optical image  
3. The system combines SAR and Optical information to estimate height  
4. Results are shown in a **table and chart**
